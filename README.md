**Checking / Installing**
Check if node is already installed:
	```
        node --version
    ```

If not, download on this link:
	```
        https://nodejs.org/en/ 
    ```

NodeJS Version
	```
        v6.9.4
    ```

Install packages:
	```
		npm install
	```

To Test. Go to repository root directory and run this command.
    ```
        node app.js
    ```

Send SMS
	```
        http://localhost:3000/
    ```
